## Crear instancia de S3

1. El nombre del bucket debe ser único dentro de una partición. Una partición es una agrupación de regiones.
2. Actualmente, AWS tiene tres particiones: aws (regiones estándar), aws-cn (regiones de China) y aws-us-gov (regiones de AWS GovCloud (EE. UU.).
3. Debe tener entre 3 y 63 caracteres.
Debe constar únicamente de letras minúsculas, números, puntos (.) y guiones (-). Se recomienda evitar el uso
4. de puntos (.) en los nombres de los buckets, excepto en el caso de los buckets que solo se utilizan para el 4alojamiento de sitios web estáticos.
5. Debe comenzar y terminar con una letra o un número.
6. Después de crear el bucket, no puede cambiar su nombre.

## Alojar sitio statico
1. Hacer click en la instancia creada>propiedades>alojamiento de sitios estaticos>editar

## Acceso a S3
    Por defecto AWS deniega cualquier intento de acceder a una instancia de s3
1. Hacer click en la instancia creada>permisos>Bloquear acceso público>editar

## Politica de bucket

1. Hacer click en la instancia creada>permisos>politica de buckets>editar

    ha editado la configuración de S3 Block Public Access para permitir el acceso público al bucket, agrega una política de bucket para otorgar acceso público de lectura al contenido de su bucket. Cuando concede acceso de lectura público, cualquier persona en Internet puede acceder a su bucket. Para llevar a cabo esta tarea, agregue la acción s3:GetObject a su bucket de S3 y establezca el valor Principal en *, que representa a cualquiera.

    Considerar: Es muy importante planificar qué contenido debe o no permitir el acceso público. Por ejemplo, los archivos que contienen información de identificación personal (PII, por sus siglas en inglés) o información de salud que está cubierta por la Ley de Portabilidad y Responsabilidad de Seguros Médicos de EE. UU. (HIPAA, por sus siglas en inglés) no deben ser de acceso público.

    ejemplo: para web statica, yourinitials-website-zipcode: bucket name

    {
        "Version": "2012-10-17",
        "Statement": [
            {
                "Sid": "PublicReadGetObject",
                "Effect": "Allow",
                "Principal": "*",
                "Action": "s3:GetObject",
                "Resource": "arn:aws:s3:::yourinitials-website-zipcode/*"
            }
        ]
    }
